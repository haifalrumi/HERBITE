#  HerBite – Healthy Recipe Web Application

## Overview

HerBite is a web application designed to make healthy eating more accessible by providing users with a collection of nutritious recipes. Users can create an account, browse recipes, save their favorites, and manage their personal profiles through a clean and intuitive interface.

---

## Project Objectives

- Encourage healthier eating habits.
- Provide easy access to healthy recipes.
- Allow users to personalize their experience through user accounts.
- Offer a simple and interactive platform for managing favorite recipes.

---

## Features

- User registration and authentication
- Browse healthy recipes
- View detailed recipe information
- Add and remove favorite recipes
- User profile with profile picture

---

## Technologies Used

- HTML

- JavaScript
