HerBite

HerBite is a web application that helps users discover healthy recipes, create an account, and manage their favorite recipes through a simple and user-friendly interface.

Features

* User registration and login
* Browse healthy recipes
* View recipe details
* Add and remove favorite recipes
* User profile with profile picture

Technologies Used

* HTML
* CSS
* JavaScript
* PHP
* MySQL
* AJAX

Installation

1. Clone the repository.
2. Move the project to your web server (XAMPP or MAMP).
3. Create a MySQL database and import the SQL file.
4. Update the database settings in config.php.
5. Start Apache and MySQL, then open the project in your browser.
